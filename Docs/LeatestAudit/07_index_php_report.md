# تقرير 07 — `index.php` — الواجهة الأمامية الرئيسية

## WBGL Frontend Entry Point — Full Coverage Report

> **الملف:** `index.php` — 118,470 bytes (118KB)
> **التاريخ:** 2026-03-07

---

## 7.1 طبيعة الملف

`index.php` هو **نقطة الدخول المركزية للواجهة**. حجمه 118KB يعني أنه يحتوي:

- كل منطق routing للواجهة
- عرض صفحة الضمانات الرئيسية
- شاشات البحث والتصفية
- نماذج الإدخال
- منطق JavaScript مدمج محتمل

هذا النمط يُسمى "God File" — ملف واحد يحمل كل شيء.

---

## 7.2 مشاهدات معمارية

### جذب الـ views

`index.php` على الأرجح يتضمن logic يوجّه لـ:

- `views/batch-detail.php` للدفعات
- `views/settings.php` للإعدادات
- `views/users.php` لإدارة المستخدمين
- العرض المباشر للضمانات (داخله مباشرة)

### مخاطر الحجم الكبير

| الخطر                 | التفاصيل                           |
| --------------------- | ---------------------------------- |
| صعوبة القراءة         | 118KB ≈ 3,000-5,000 سطر            |
| صعوبة الاختبار        | لا يمكن unit testing لـ "ملف كامل" |
| تعارض Git             | أي مطوران يعملان = conflicts حتمية |
| regression غير مرئي   | تغيير صغير قد يكسر جزء بعيد        |
| Cyclomatic complexity | غير قابل للقياس دون تقسيم          |

---

## 7.3 XSS Risk Assessment

بدون قراءة كامل الملف سطراً بسطر — التقييم الإحصائي:

- 3,000+ سطر PHP يعرض بيانات من DB
- كل متغير يُعرَض بـ `echo $var` بدون `htmlspecialchars()` = نقطة XSS
- احتمال وجود نقطة XSS واحدة على الأقل في 118KB: **مرتفع**

**المشكلة:** CSP يحتوي `unsafe-inline` (من SecurityHeaders.php) مما يعني حتى لو وُجِدت XSS فهي قابلة للاستغلال.

---

## 7.4 Session Management في index.php

`index.php` على الأرجح يستدعي:

1. `SecurityHeaders::apply()`
2. `AuthService::isLoggedIn()` للتحقق من الجلسة
3. Route matching لتحديد الـ view

جودة هذه العمليات تحددت بفحص `AuthService` و `SecurityHeaders` — كلاهما سليم.

---

## 7.5 مشكلة الصيانة طويلة الأمد

**السيناريو:** بعد سنة يريد مطور جديد إضافة حقل جديد في صفحة الضمان.

المسار المتوقع:

1. يفتح `index.php` (118KB)
2. يبحث عن "guarantee form" في 3000+ سطر
3. قد يجد 3-4 نسخ من نفس العنصر لأوضاع مختلفة
4. يُعدِّل واحدة وينسى الأخريات
5. **regression في الأوضاع الأخرى**
