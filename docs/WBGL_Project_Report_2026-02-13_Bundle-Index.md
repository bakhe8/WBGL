   - تدقيق/قانوني → Audit-Legal
   - فريق تقني/تشغيلي → As-Is Technical

3. ارجع للنسخة الأصلية عند الحاجة لسياق تاريخي أوسع.

## 8) قاعدة الحوكمة على الوثائق

- النسخة الأصلية تعتبر المرجع الشامل.

- أي تحديث جوهري في سلوك النظام يجب أن ينعكس على:
  - النسخة الأصلية + V2 + النسخة التدقيقية + النسخة الوصفية + ملحق المصطلحات.
